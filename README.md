# TDE2 - Sistema de Gerenciamento de Tarefas
Trabalho prático da disciplina de Raciocínio Algorítmico usando POO em Python.

## O que é
Um sistema simples feito no terminal que permite criar, listar, atualizar e excluir tarefas. Cada tarefa tem título, descrição, status e data de criação.

## Como rodar
Precisa ter o Python instalado. Depois é só rodar:

## Arquivos
- `gerenciador_tarefas.py` - o código do sistema
- `documentacao_tarefas.docx` - documentação do trabalho
